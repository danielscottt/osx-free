Free
====================

_Because OS X doesn't have a_ `free` _command_

```bash
$ free
Usage:
  free [options]
       --watch, -w:   watch command
          --gb, -g:   view in GBs
          --mb, -m:   view in MBs
  --show-units, -s:   show unit size
     --version, -v:   Print version and exit
        --help, -h:   Show this message
```

##Installation

```bash
$ gem install osx-free
```
